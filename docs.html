Install Hentoki CLI

The commands below show how to install the CLI on different platforms.

Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install Hentoki-cli

NPM

$ npm install Hentoki-cli

Yarn

$ yarn install Hentoki-cli
